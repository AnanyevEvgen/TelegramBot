### _TelegramBot_, створений як контрольний проект під час навчання на курсі від JavaRush.
1. Виводить цікавий випадковий факт.
2. Відповідає на поставлене запитання.
3. Надає можливість поспілкуватися з відомою особистістю.
4. Надає можливість перевірити свою ерудицію у вікторині.
5. Детально описує, що зображено на завантаженому зображенні.
6. Пропонує твір мистецтва на основі критеріїв користувача.
